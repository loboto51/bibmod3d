Description :

~~~
An 8mm scale Orbital Lander / Dropship compatible with GW LegionsImperialis / AdeptusTitanicus / Epic.

The compartment is sized to accommodate:
- 8no. x Leman Russ tanks
or
- 4no. x Malcador tanks
or
- 2no. x Baneblade tanks
(or similar size models)

Design inspired by building module landers from original Dawn of War game.

Includes openable front ramp, and engine pods with slots for magnets (5mm x 1mm) which will allow them to rotate freely. Alternatively engine pods can be glued in place.

Two options included for the landing legs.
Some rooftop components print separately and can be mounted in different positions.
~~~
