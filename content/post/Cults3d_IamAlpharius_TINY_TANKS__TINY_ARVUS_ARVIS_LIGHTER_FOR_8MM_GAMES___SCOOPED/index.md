I do not recall where I grabbed this Arvus Lighter model from.  I think it was an unattributed free source (jimmyjimjim reposted it).  It came from somewhere and when I find out where, I'll add attribution.

Some idiot STL Stasi type will undoubtedly have an issue with that.  Shut up.  Either help me or go take a giant leap at a rolling donut.

Now - to the important stuff.  I've not test printed this one but it is a very simple model in very good shape with no hollows to trap resin and the walls are pretty thick.  The magnet slot (5mm) is inside on the roof and set at about a 3-5 degree angle down (can't remember) so the Arvus Lighter have a nose down look to it like a modern helo at cruising speed.  

I hollowed and opened up the internals so that it can be properly UV light cured and left the door open in case someone wants to put a gunner back there :)

The wingspan is about 40mm, so I think it looks about right compared to what I have seen for the AI/LImp one.  Resize as needed but be prepared to use a smaller magnet
