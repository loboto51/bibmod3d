This is my take on Stormhound Assault Walker, Proxy for a Stormtalon, based on Spikey Bits Original Stormhound Conversion.

The Legs are from Seb's Awesome Armor-Bearer, and were scaled to fit this design. It's a single file, so, if you need them separate, please hop over to Seb's to get them and scale them up 1.5

File names indicate how many parts need to be printed.

Enjoy and support the plastic kit makers.

If you like my models, consider leaving a tip! :)

Edit 19/05/19 - Corrected an interference between the fuselage pin and the legs. 
Edit 05/04/19 - Changed the connection from the fuselage to the Legs by 2mm in height, to give the thrusters a bit more clearance. Also changed the name of the Engine nozzle. It indicated you'd need 2 when in fact only 1 was needed.
