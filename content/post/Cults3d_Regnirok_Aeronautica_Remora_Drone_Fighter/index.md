Description :

~~~
This is a remix of the Tau Remora drone fighter in 28mm scale, but I've:
 
1) Assembled the model so it will print as one piece
2) Scaled it down to roughly AI Scale
3) Added a socket mount to the underside that is compatible with both the clear AI flight pegs as well as my Tall AI Flight Stand models (https://www.thingiverse.com/thing:3863230).

Package includes both an unsupported and supported version.  I placed my supports for printing with Siraya BLU resin on an original Anycubic Photon, so the supports should work with nearly all other resins and printers.  Models will need a little cleaning on the underside.
~~~
