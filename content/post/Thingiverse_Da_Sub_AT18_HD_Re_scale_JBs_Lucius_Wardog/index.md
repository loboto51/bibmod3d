~~~
An AT18 Rescale and Remix of Jimbeamz Lucius Wardog. I removed some iconography and modded a couple of bits plus merged some and added pre-done holes in the torso. Its not perfect but it should work and is fairly close in height to one of the official plastic Mars Pattern Kits. Please note I have not done the Plasma or Flamer weapons yet. I will upload them when they are done.

Notes: 
- Primarily aimed at being printed on a DLP Printer
- Parts are not merged so if you want to repose the legs feel free.
- If GW release a plastic version of this I will be removing it. At the moment it is an unavailable kit so I'm posting this in good faith for free.
- Do not request re-ups of the original 40k Scale file. I won't be doing that.


Edit 05/02/2020: Added remixed Turbo Plasma and the Plasma Canisters for it.

Edit 06/02/2020: Found Three Rivets I missed removing that floated in space under the Turbo Plasma, added a fixed version of the file. If you downloaded on day one grab the new file.

Edit 08/02/2020: Noticed some weird artifacting in the plasma gun when you slice it. Specifically there appears to be an invisible block extending in the core of the weapon into the socket for the arm mount. I'll have to work out which addition is causing this and clean it up manually. If you print it out before then you will need to dremel or drill out the socket. I'll try and fix this when i get some time off work.
~~~
