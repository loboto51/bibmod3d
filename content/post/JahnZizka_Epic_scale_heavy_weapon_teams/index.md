Description :

~~~
Add some firepower to your infantry companies.
Included many poses (firing, reloading, carring the weapon, etc.) and ammo boxes too!!
Weapon options included: Heavy bolter, autocannon and lascannon.

All unsupported and supported elements are included in their respective chitubox files.
Bases included ready to print flat on the base (0.04 layer and 4 bottom layers for optimal results given the raiser height)

UPDATE: Included squad 1 and 2 with all 3 weapon options for easier use.
Reuploaded given Thingiverse inability to index properly.

Sources:
6/8mm Empire Star Army, Acadian Infantry by Miffles_Makes https://www.thingiverse.com/thing:3746245
Lascannon (Heavy weapon team) by Solutionlesn https://www.thingiverse.com/thing:3776215
Autocannon (Heavy weapons team) by Solutionlesn https://www.thingiverse.com/thing:3781341

Updated: Test print resulted in some errors due to slicing software (Photon Workshop) adding a column of material due to an open poligon mesh (all other software sliced it perfecly though). Reuploaded all the stls with the fix and added a ZIP file with all the elements just in case Thingiverse redirect the Download All button to a 404 error again...
~~~
