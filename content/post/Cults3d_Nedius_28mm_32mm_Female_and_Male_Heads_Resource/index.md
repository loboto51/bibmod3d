Description :

~~~
For a variety of projects I've got on, I needed a set of human heads to add to models I intend to sculpt for 3D printing. I couldn't find any I like, so have made up a collection of heads I've created by scratch sculpting, using Blender Add-ons, etc. None are 'stock' generation.  Each has been scratch sculpted, re-sculpted from add-on generated blank faces, modified, re-shaped etc by me to meet the needs of my projects for 28mm - 32mm scale miniatures. 

I really struggled to find any free base heads I could use as a starting block to model onto, so am putting these here as a resource for any other amateur enthusiast designers like me who need a start until our scratch-sculpting talents get to where they need to be!

If you do use them in your own projects, would love to see them.
~~~
