~~~
This is a re-mix of Smight's lovely Vectored thrust VTOL (https://www.thingiverse.com/thing:4639577).  I've added a peg hole to fit my hex flight stand (https://www.thingiverse.com/thing:4766450) as well as included a version of the engines with exhaust that won't need the flight peg.  With the exhaust version, you'll need to print it twice with one copy mirrored and print the main body of the VTOL from the original design. 

Please see the instructions at https://www.thingiverse.com/thing:4766450 for how I recommend printing the stand.

I had great success printing the aircraft itself on my Ender 3 by angling it up 45 degrees, which greatly reduced layer lines and gave it a clean underside.

If you like this, please consider a tip to Smight: https://www.thingiverse.com/smight/designs
~~~
