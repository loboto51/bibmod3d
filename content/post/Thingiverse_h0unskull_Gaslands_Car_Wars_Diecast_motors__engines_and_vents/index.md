Description :

~~~
Simple add-on parts to modify any diecast car for any postapocalyptic racing game. Scales well with ~1:64 original Matchbox scale. Primarily designed for Gaslands but can be used for any other games as well.

This thing contains bits to pimp or assemble engines. Engines and air inlets are printed separately and can be combined as wanted to get more variety.

**Updates:**
**18-07-31:** new standalone motor block and addon motorblock added (can be used as single or put on top of an existing one.

This thing is the third release to get the original thing https://www.thingiverse.com/thing:2889336 sorted/grouped and contains a few new elements. Hopefully this will make future releases more easy.

Additional elements will be added in a (not so) timely manner...
