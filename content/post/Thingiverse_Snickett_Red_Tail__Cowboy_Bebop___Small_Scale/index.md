Heavily modified from the wonderful model by Taffgoch on the 3dWarehouse:
https://3dwarehouse.sketchup.com/model/5990c977dcd97dee37ddfe82c734947d/Red-Tail-Cowboy-Bebop

Adopted for small scale prints, so some details are simplified.

Originally created to fit in with the 1:270 scale of the X-Wing Miniatures game. However, the Red Tail feels entirely too small at that scale. As such, I've uploaded a 1:210 scale version as well; this should give the cockpit the same size as a TIE Fighter cockpit.
