6\8mm scaled Draugr Pattern Gunship for "Galactic Crusaders"(c) Moonjam.
Scaled, cleaned geometry to get good hollowing, rearranged some elements, changed turrets from heavy jet-guns to laser cannons. 

Huge, fits Photon S plate vertically. Probably not intended for FDM printers.

Haven't printed it myself yet, but Chitubox says 18 hours for my print settings. 

Remix sourcing still down, so I'll try not to forget add sources when it would be back online. Here them are:

Medusa Pattren Stormbird https://www.thingiverse.com/thing:4000035
Tiny Tank Classic MBT https://www.thingiverse.com/thing:4397478 for laser cannons


15/10/2020 - Added chitubox file with supports, adjusted after print of first version. 16h35m printing time at my print settings (0.04 layer, 6s\layer, 60s bottom layers, 40mm/min retract for both).
