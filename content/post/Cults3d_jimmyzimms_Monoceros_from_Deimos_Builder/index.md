Note: Moved from my Thingiverse profile since the site is perma-broken it appears

Provides a version of Woddish's excellent Tiny Tank Classic Mbt (now gone), KMFP proto-rhino concept, and a Scorpios launcher of unknown source, and a radar from Chris John, that have been sliced into major parts that are enabled to be quickly swapped and edited for customization by changing visibility of components between different options for hatches, side doors, symbols/icons, and crew. The eventual end game would be the ability to build any Deimos based Monoceros chassis vehicle with this tool.

This is intended for designers and modelers to product their own faction customized vehicles from in a standardized and high quality form.

If you like what I did and want to leave a tip it would be appreciated.
https://paypal.me/jimmyzimms

Edit 20220526: Found some flipped normals I missed and a couple of positioning errors so corrected. Since 3D Builder gets annoyed about nonmanifold object and does the wrong thing occasionally I've updated all the parts, even if it shouldn't matter.

Edit 20230112: Added a slicer for the side doors to the blender file that can be used by designers to make form fitting symbols from that can be set into the hatches.

Edit 20230426: Added support for KMFP's nifty 3rd edition style Whirlwind Launcher

Edit 20230519: With permission, incorporated thewraeththu's better Scorpius into the builder.

Edit 20230908: Updated the Scorpius launcher to have an upper deck based on the Deimos predator. This better matches to the updated plastic kit from GW.

Edit 20240117: Minor cleanup

Edit 20240518: For those that like rivets and bolts, two sets (based on the geometry of each glacis option) have been added to adorn your tanks with tons of extra! Thanks TC63 for the contributions!

Edit 20241027: Added in crew from the Rapax From Deimos builder here.
