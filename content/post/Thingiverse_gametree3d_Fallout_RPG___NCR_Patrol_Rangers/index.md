Description :

~~~
Miniatures designed for and scaled to 15mm for use with Fallout Wasteland Warfare RPG. 

To access files individually load the .stl file in Microsoft 3D Builder and repair the file (takes about 5minutes) then select ungroup.

Alternatively you can load the .3mf file in Microsoft 3D Builder and select ungroup; this method will allow you to manipulate the models from their pieces and you can swap around heads, weapons and poses to your desire.

The base bodies are from reddawnminiatures
The arms and flag are from ziocru on cults 3d.
The heads, shotgun and carbine are taken from a garrymod addon for fallout rpg.
~~~
