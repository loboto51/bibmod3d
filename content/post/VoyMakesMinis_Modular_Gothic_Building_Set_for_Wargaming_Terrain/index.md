Description :

~~~
Here's my set of modular building elements that allows you to build gothic-like structures for your wargaming terrain. I designed it for FDM printers.

The initial idea was to be able to create simple gothic-like walls in both undamaged and damaged form, so this is the reason why there are no floor tiles just yet.

There are no fancy connecting mechanism - use the glue, Luke! I prefer to use quick-drying basing glue. 

**1.0v - Initial Release**

- Includes 27 wall and column modules.

**Updates**

- 18-07-2023: after some more printing, I can confirm that resizing the modules down to 80% still works great. I can also confirm that the modules print easily on a resin printer - with no supports, placed directly on the build plate. To save resin, you can hollow the columns (don't forget to add the drainage holes).
~~~
