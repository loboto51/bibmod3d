Description :

~~~
Some time ago I was felt very inspired by the 40K scale imperator titan from Zammerak  you can find here in Cults, so I started remixing my own head variant, more close to the titan in the Titandeath cover, a design closer to the original Epic imperator model and featuring a full detailed interior cockpit. I used for the remix the head from the zammerak´s titan and a low poly faceplate model here in cults, apart from hundreds of bits from Cults and Thingiverse for the interior detailing. 
The size is calibrated to lore 40K scale titan, of about 95-110 cms. It´s compatible with my Imperator model and Grant´s model but it will need to be rescalated about -25% 
In the plans if fits OK but when I´ve printed it needed a lot of sanding in the cap part :-( Version 2 features a simplified cap with better fit.
The interior features complete cockpit with monitors, controls, a holographic display and a hiden room bellow with as little chapel and 2 servitors. Hope you like it :-)  Complete crew (2 moderatii and 1 princeps) in the donators rewars folder.

This model is remixed from this models:

- For the base I used zammerak´s titan head, you can fin the complete titan in Cults free Here: https://cults3d.com/en/3d-model/game/titanic-church-delivery-system
- For the faceplate I choose this model from Fantabuloustexan, Here free in Cults3D: https://cults3d.com/en/3d-model/various/walkin-church-titan-head
- And hundreds of bits from Thingiverse and Cults3D 

Updates/Changelog: 

25-11-22  Added 3 neck versions for using in different imperator bodies sizes and a instructionsTXT
4-12-22 Added new head version 2! size adapted to my Imperator body (AT scale X 4), more detailed and with a simplified cap with better fit.

Donations to the artist welcome!
