I wanted some space goblins for playing [Grim Horizons](https://www.wargamevault.com/product/336867/Grim-Horizons--Adventures-Told-Through-Combat) so i made this remix from some available models. The figures are approximately 20mm tall without any base so should get scaled if they are not being used as small characters.

If you enjoy these models please consider [tipping me a coffee](https://www.paypal.me/dotplot1) to encourage creation of new designs and improving existing ones.

Note: all the backpacks, googles, grenades, and other bits are separate shells which can be easily removed using meshmixer or other modeling software if you don't want them.

Update: the download all button is now working so the temporary ZIP files have been removed.
