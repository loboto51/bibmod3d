Description :

~~~
Imbued with excess and pain, the imagination of the dark mechanics forged the Rapture class Heavy Stalker. 
Hunters of titans, they bound and leap across the battlefield after their prey.

This kit includes everything you need to build a Rapture Heavy Stalker in 8mm or AT scale.
The main body measures about the height of the standard Heavy Stalker with the tail adding extra height.
This can be built in any number of ways, the kit is extremely flexible.

The kit includes, amongst standard components:
- 3 different heads, with and without tentacles
- 2 tail poses
- 2 different tail weapons
- 2 different arm weapons

Four files include:
1. Components: individual components to print and pose as you desire. Includes the supported folder.
2. Poses: a number of arms and leg poses to make assembly quicker. Supported versions included.
3. Complete: some example builds which you can use as an example or print as a single object. Supported versions included.

The model has been made to go on a 60mm bases and this has been included.


All files come supported and unsupported.

Pictures of models are test prints.
~~~
