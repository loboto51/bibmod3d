I can also upload a splittable mesh if desired, for easier remixing.

If anyone has SVGs/stls/meshes of filigree/ornaments (see all the details here: https://cdn.shopify.com/s/files/1/1189/6522/products/Storm_Shield_C_1024x1024.JPG?v=1556292533) That would make it way easier to make more shields!

Edit 1: Added grip for regular sized guys.  Both of the grips should be compatible with the modular builders for regular sized guys and bigger terminating guys (as those are where the hands came from).
