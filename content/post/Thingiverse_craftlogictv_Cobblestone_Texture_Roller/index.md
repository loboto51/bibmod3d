This is the cobblestone roller that I designed on my YouTube channel. I've had several requests for this roller, so here it is! I have successfully printed this on my Elegoo Mars with supports on the bottom just to increase the surface area on the build plate. A raft would likely work just as well. I have not tried printing on an FDM printer but I imagine it will need more supports for that. I'd love to know how it turns out if anyone tries to print on FDM. I've included some pictures of a model I build that utilized this roller so you can see how it turns out when used in a project. I hope you find it useful!
