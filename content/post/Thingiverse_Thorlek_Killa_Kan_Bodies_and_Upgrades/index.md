~~~
I love the original model by VidovicArts and printed a couple of them, but decided I wanted some more body and weapon options.

I spent an ungodly amount of time in TinkerCAD modifying the original parts to print easier and allow more options.

I figured some other people might like these parts.
Hopefully the original creator doesn't mind I've posted these parts, if you do let me know.

The arms have a flat bit on the shoulder/ball joint, so if you put that flat section on the build plate, you can print them without supports.

Everything else just print how the file is orientated, and use supports on the bottom of the bodies, all the weapons should print without support.

I'll post some photo's when I've done painting some.
~~~
