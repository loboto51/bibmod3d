Description :

~~~
This model is designed for use in Games Workshop's 6mm scale table top wargame, Epic 40,000 (and precessors/successors). They have since abandoned all support for Epic, so I modeled this in Autodesk Inventor for any avid wargamers.

I don't believe there are any official rules for the Minotaur in Epic 40k, but I really like the look of this beast compared to the Basilisk. It adds a bit more variety, or allows the player to field a 30k-themed force.

Epic has been re-released as a titans-only game called Adeptus Titanicus. It is 8mm scale instead of 6mm, so I suggest that anyone wanting to print this for AT scenery or to play 8mm Epic should upscale it by 133%.
~~~
