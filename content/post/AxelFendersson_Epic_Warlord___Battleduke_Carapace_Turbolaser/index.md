Description :

~~~
This is a shoulder/carapace mounted Turbolaser Destructor (twin turbolaser) for use with Stroganoff's Battleduke Warstrider model.

It's a mashup of this Apocalypse Rocket Launcher by totenpf:
https://www.thingiverse.com/thing:4814519
...the turbolaser barrels from this Wardog:
https://cults3d.com/en/3d-model/game/lucius-wardog-titan
...and part of the Laser Blaster model that comes with the Battleduke
https://cults3d.com/en/3d-model/game/epic-scale-mars-warlord-titan

I took all the bits, chopped them around, scaled them to fit and mashed them together.

It should fit exactly onto the Battleduke model if you print them both at the same scale. If you want to use it with an AT-scale model you'll need to scale it up a bit; probably about 125% but I don't know exactly as I haven't tried it and I don't currently have any AT models to measure.

The model is slightly asymmetrical. I think it works better on the right shoulder; mirror it for the left.
~~~
