Description :

~~~
This is a set for creating a city building as a terrain for any wargame in 1/285 (6mm) scale. The set allows you to assemble both an intact (undamaged) buildings and a destroyed ones (ruins). You can assemble a building of any height you want by printing the required number of intermediate floors.


There is one option with square windows (marked with number 1) and the second with rounded windows (marked with number 2).


To make it easier to paint the windows, I made it a separate piece that can be painted in advance (for example, with spray paint) and inserted inside the walls.


I also made a set of interior parts in case you don't use windows or want to make glass from transparent material.
~~~
