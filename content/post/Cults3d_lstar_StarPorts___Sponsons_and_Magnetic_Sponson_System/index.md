Description :

~~~
All your sponson needs, all in one place!

A central collection so I don't have to update a half dozen packs when modifying the sponsons, and the Universal Sponson Pin needed for the Magnetic Sponson System.

**Wait a minute, Magnetic Sponson System?**

The Magnetic Sponson System is an easy way to swap out sponson weapons. Want to change up loadouts between battles, or need to replace a snapped barrel? Just slide the old sponsons out and new ones in!

Each pin has slots for a normal sponson weapon and a 2x1mm neodymium magnet.

The pins are symmetrical, so there is no need to mirror them. Just remember the rivets point backwards, and make sure to decide on which polarity to use for the left and right pins!

**Note:** This is currently compatible with the Pred-8R, SIG-RN, and Crete-S packs.
