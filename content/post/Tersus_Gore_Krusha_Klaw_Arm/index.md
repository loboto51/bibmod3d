When building Gore Krusha Mega Walkers, greenskin Meks sometimes opt to build massive klanky klaws as melee weapons instead of the often-seen giant saws.


FOR BEST QUALITY PRINT, PLEASE READ ALL(!) OF BELOW:

This highly detailed Klaw Arm is compatible to my Gore Krusha's Mega Walker
https://cults3d.com/en/3d-model/game/gore-krusha-mega-walker

I REALLY recommend a layer high of 0.050 or lower for all parts to get the quality I intended. 
Please see the pics of my print, those are all in FDM (0,05mm layer height)!

I tried to keep the need of tree supports low, but I recommend it to use them on some parts - decide for yourself & make sure to set the tree support z-distance to a multiple of 2 (layer height) for optimal removal (keep in mind: layer height 0.05 to 0.075(!) )


----------------------------------------------------------------------
-------------- EXCEPTIONAL 'fings for this model: --------------------
----------------------------------------------------------------------

---------- MOVABLE PARTS ----------
- Most pivot points are actually functional, pose the arm as you want!
- I have included a functional hydraulics option for the klaw, so it can move once assembled.
 -> You need a well calibrated printer to get this hydraulics working. Print all *FlexHydraulics*.stl with wallcount = e.g. 99 to get a massive print without infill.
 -> For the FlexHydraulics pins you need to cut of ca. 10mm of regular 1,75mm filament. (I did not test this in resin, but I guess you could print them)
 -> As an option to the FlexHydraulics there are two fixed versions. Have a look in the parts.
- Print all PINS with wallcount = e.g. 99 to get a massive print without infill.

---------- CABLES and WIRES ----------

KlawArm_03 Bits come with an option to glue in some wires (or filament). 
If you use some 0,75mm² or 0,5mm² electrical wires the arm could still be movable :)

---------- Arm on the LEFT or RIGHT? ----------
You can print all parts normally except the shoulder pauldrons. (because of the flat printing side)
-> You have to mirror both pauldrons if you want the arm on the left side of your model (from your point of view)

---------- ASSEMBLY ----------

If you need instructions - just open all files in 3D Builder or e.g. Meshmixer. There you can look where they need to go.
