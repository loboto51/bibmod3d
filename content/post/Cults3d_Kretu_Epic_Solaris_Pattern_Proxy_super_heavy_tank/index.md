Description :

~~~
A proxy for super heavy tank of support troopers from solar system. Includes rivets. So many rivets. Also recessed armour panels, golden trim, re-designed co-ax autocannon and other embellishments. Bottom is hollowed out for reduced resin usage. Intended for 6-8 mm wargames and such. 

Test printed on Anycubic Photon Mono 2 at 35 mic layer, you can see the results on the photos.  I`ve succesfully test printed all the turrets, both HB hulls and AC hull, have not yet printed the HF version. 

Credits:
 - Woddish excellent  biggest MBT  served as the base model for this remix
- Mkhand_Industries limp russel/sealant upgrades - I`ve used a gun shield for the demolishing cannon, as well as all the new type sponson weapons
- MrOrangeJumper - a commander in the hatch from the solar troopers command

Updates:
13/01/24 - two rivets didn't apply properly to mesh, so I fixed them.  
22/01/24 - added hull with new type sponsons that point both ways, 3 weapon options included, also turret wariant with short heavy cannon and another with commander in the hatch
~~~
