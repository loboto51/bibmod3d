A revise of the models made from 3D Desktophero which I took down because they had printing issues and lacked details. These models have been designed for and scaled to 15mm for use with the Fallout Wasteland Warfare RPG rule set. To access files individually, open the file in Microsoft 3D Builder and select ungroup. 


The only files missing from this remix of note are the base figure pose models, file named outlaws, which were pulled from Cult3D by Wasteland Trader, who no longer has an account on the site. Additionally, the trail carbine/revolver is from cults3d; Interstellar Army Wasteland Weapons from MKHand_Industries.
