Description :

~~~
A true-scale 6mm version of the M'khand Pattern Lemoine Russel Main Battle Tank. Some details have been removed and others altered to make them more viable at this scale.

Perfect for playing the various editions of Epic and as base toppers for Adeptus Titanicus 2018.

Turrets and hulls have 2.5x1.25mm holes for magnetising them to the bodies.

2021-02-17: Updated body & turrets no longer compatible with previous epic scale leman russ tanks. Added new main weapons and trench rails. Reduced volume of the bodies.
2021-11-07: Added a zip with Annihilator, Punisher, and Vanquisher turrets.
2022-07-02: Replaced old files, replaced zip archives with .3mf files. Added a Phaeton-pattern demolisher cannon.
2022-09-04: Added turrets without the side-mounted optics.
2022-09-15: Added a builder for use with meshmixer that has different side-sponson options.
2022-09-17: Fixed some previously unseen holes. Added new middle sections to the meshmixer builder that have sleeker hull sponsons as well as a new 'Beta' pattern.
2022-10-01: Added new twin-laser turrets
~~~
