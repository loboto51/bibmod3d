The Skrappin' Mega Bot is a stompin' armored Mech capable of moving objects without any efforts using it's powerfull trakktor kannon! Dead 'ard 

killy saws tear apart every enemy getting to close. Once the Skrappin' Mega Bot krushed some enemy vehicles, turing those into piles of skrapp, it 

can do some quick n' dirty fixin' jobs on allied vehicles or mechs.


FOR BEST QUALITY PRINT, PLEASE READ ALL(!) OF BELOW:

Beneath my Krumpin' Mega Bot this is one of my most detailed models until now!

To print this monster, you need time or a resin printer ;-)
I REALLY recommend a layer high of 0.075 or lower for all parts to get the quality I intended, while designing this model - please see the pics, those 

are printed in FDM (<=0,075mm)!

I tried to keep the need of tree supports low, but I highly recommend it to use them - make sure to set the tree support z-distance to a multiple of 2 

(layer hight) for optimal removal (keep in mind: layer hight 0.05 to 0.075(!) )

For a few bitz I added a resin version with "_Resin" in the file name - unlike the FDM versions those don't have a flat bottom. (I have not test printed 

these files, but they should be good to go!)

The Trakktor Kannon can be a variant of an actuall (rattling) Kannon, as it lifts up 'eavy objects and throw them with a high velocity to enemy gitz - 

less explosiv, more smashin' - same damage! ;-)

---------- COMPATIBILITY ----------

Most parts are compatible to my Krumpin' Mega Bot, so you can build your variant as you like :)
https://cults3d.com/en/3d-model/game/krumpin-mega-bot

---------- MAIN BODY ----------

You CAN use two 10x4mm Neodym magnets for the junction "hip <> main hull" to get a rotatable/removable body!
As an alternative to the magnets print the file "ScrappinMegaBot_Hip02.stl"

---------- LEGS ----------

The legs are unique, there is a individual model for the right / left leg. Those are mirrorable / interchangeable if you want.

---------- Over all printing recommendation! ----------

Many parts have a "flat surface" - face those to the build plate.
