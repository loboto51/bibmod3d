I wanted some alien raiders for playing [Grim Horizons](https://www.wargamevault.com/product/336867/Grim-Horizons--Adventures-Told-Through-Combat) so I made this remix from some available models. The figures are approximately 30mm tall without any base so should get scaled if they are not being used as medium characters.

Modular kit includes rifles, carbines, shotguns, pistols, melee weapons, grenades, and various special weapons.

Note: users will need to mirror some arms and Zip files are used for organisation.

If you enjoy these models please consider [tipping me a coffee](https://www.paypal.me/dotplot1) to encourage creation of new designs and improving existing ones.
