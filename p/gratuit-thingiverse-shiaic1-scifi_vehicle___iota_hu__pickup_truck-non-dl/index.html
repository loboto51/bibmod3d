**Iota-HU, Pickup Truck**

This file is another remix/remaster of a file made by [TheDude21](https://www.thingiverse.com/thedude21/designs) (check him out and give him a tip, if you like). Much like the previous Glade-S, I very much liked the core Cyberpunk stylings of this vehicle concept and thought that it would make a nice adaption into a 'lower' tech level universe setting as a civilian hovercraft vehicle at the 6mm scale setting.

**About the Design.** Despite the superficial resemblance's to the original design, this project actually took quite a lot of time to remaster, because I wanted to remove as many of the 'hard-angle' edges as possible. This should make the model much better to orientate for supports and 3D print on a resin system. (zoom right in on the file to compare it with the original, if you like!)

**Other Designs.** This model set is intended to be compatible with the other 6mm (true-scale) and Hex(33) (hex scale) SciFi terrain sets that I have released on [Cults 3D](https://cults3d.com/en/users/Shiaic3D/creations) and other locations, [GumRoad](https://shiaic3d.gumroad.com/), [Printables](https://www.printables.com/@Shiaic3D/models) and [Thingiverse](https://www.thingiverse.com/shiaic1/designs).

Please search my other designs for more models of a similar type and theme.
