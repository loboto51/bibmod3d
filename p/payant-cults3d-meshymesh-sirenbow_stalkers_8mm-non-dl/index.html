Imbued with excess and pain, the imagination of the dark mechanics forged the Sirenbow class Stalker. 
Walking guns, they stalk across the battlefield picking off targets before they know it.

This kit includes everything you need to build a Sirenbow Stalkers in 8mm or AT scale.
The main body measures about the height of the standard Stalker.
This can be built in any number of ways, the kit is flexible with the legs and body being posable.

The kit includes, amongst standard components:
- 2 different feet
- 3 tail weapon options

Four files include:
1. Supported: individual components to print and pose as you desire.
2. Unsupported: as above but supported.
3. Complete: some example builds which you can use as an example or print as a single object. Supported versions included.

The model has been made to go on a 50mm bases and this has been included.


All files come supported and unsupported.

Pictures of models are test prints.
Pictured in the background are the [Rapture Heavy Stalkers](https://cults3d.com/en/3d-model/game/rapture-heavy-stalkers-8mm)
