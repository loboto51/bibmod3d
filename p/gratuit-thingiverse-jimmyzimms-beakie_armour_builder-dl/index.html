Builder Doll for anyone wanting to have quality components for making your own Beaky Army remixes. Performed some serious fixes on Moonjam base parts and a newly constructed pauldron shell. Included is the KillMeForPrizes helmet. Legs originally from Danitblobby. Initially released as Meshmixer file since that  seems to be the lowest common denominator for people to use, it now contains a blender version as well.
